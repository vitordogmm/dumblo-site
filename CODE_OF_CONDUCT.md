# Código de Conduta

Este projeto segue os princípios do [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Padrões Esperados
- Seja respeitoso e acolhedor.
- Evite linguagem ofensiva e assédio.
- Considere impacto e contexto cultural.

## Relatos
- Abra uma issue com o tipo apropriado (bug/feature) e marque com `community` se for comportamento.
- Para abusos ou assédio, você pode abrir uma issue privada (se habilitada) ou contactar os mantenedores pelo servidor de suporte.

## Enforcement
- Violações podem resultar em avisos, suspensão temporária ou banimento das contribuições, a critério dos mantenedores.

A participação neste projeto implica concordância com este código.
